🔥The Pitch

Agent Skills are structured packages of procedural knowledge that augment LLM agents at inference time without model modification. A Skill is a SKILL.md file with instructions plus optional scripts and references — adopted across Claude Code, Gemini CLI, OpenClaw, Codex, and 30+ agent platforms.


Eight focused Skills papers appeared in a single month (Feb–Mar 2026). An audit of 3,984 Skills found 37% contained security flaws. SkillsBench revealed that curated Skills improve pass rates by 16.2 points on average, yet self-generated Skills can hurt performance in nearly a third of tasks.


The field is moving fast and needs a dedicated venue. AgentSkills '26 brings together researchers and practitioners working on Skills design, benchmarking, optimization, security, and ecosystem infrastructure.

📋Topics

Design & Authoring

Principles for effective Skills; structure–efficacy relationships; domain-specific patterns

Evaluation & Benchmarking

Frameworks treating Skills as first-class artifacts; metrics beyond pass rate

Synthesis & Optimization

LLM-based generation (GEPA); RL refinement; evolutionary approaches

Composition & Scaling

Multi-Skill effects; interference and synergy; optimal selection and retrieval

Domain Applications

Healthcare, manufacturing, scientific computing, cybersecurity

Ecosystems & Infrastructure

Registries, versioning, distribution; interoperability across harnesses

Security & Supply Chain

Malicious Skill detection; formal verification; sandboxing; supply-chain attacks

Continual Learning

Feedback-based improvement; trajectory mining for Skill discovery
